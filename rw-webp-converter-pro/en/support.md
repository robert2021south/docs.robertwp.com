# Support

This page explains how to get support for RW WebP Converter Lite and what is covered by the Lite version.

---

## Before Contacting Support

Before requesting support, please make sure you have:

- Read the documentation pages
- Checked the [FAQ](faq.md)
- Verified that your server meets the plugin requirements

Many common issues can be resolved by reviewing the documentation.

---

## How to Get Support

If you need assistance with RW WebP Converter Lite, you can get support through the following channels:

- WordPress.org support forum (recommended for Lite users)
- GitHub Issues (for bug reports and technical issues)

Please note that response times may vary.

---

## Lite Version Support Scope

Support for the Lite version includes:

- Installation issues
- Basic plugin functionality
- Bug reports related to the Lite features
- Compatibility issues with WordPress core

The Lite version does **not** include:

- Server or hosting configuration support
- CDN or cache configuration assistance
- Custom feature requests
- Debugging third-party plugins or themes

---

## Information Required for Support Requests

To help resolve issues efficiently, please include the following information when contacting support:

- WordPress version
- PHP version
- Image library used (GD or Imagick)
- Plugin version
- A clear description of the issue
- Steps to reproduce the problem (if applicable)

Incomplete reports may take longer to resolve.

---

## Pro Version Support

The Pro version includes:

- Smart PNG/JPEG Optimization
- WebP & AVIF Formats
- Retina & Responsive Optimization
- Preserve EXIF Metadata
- Conversion History & Advanced Logs
- CDN / External Storage Compatibility
- White-label Mode
- Priority Support & Early Access

If your project requires advanced image optimization, CDN integration, or responsive image handling, consider upgrading to the Pro version.

---

## Feedback & Suggestions

Feature suggestions and general feedback are welcome.

Please note that not all suggestions can be implemented, but all feedback is appreciated.

---

Thank you for using RW WebP Converter Lite.
